# Weather Prediction ML Project

## Project Overview

This is a Machine Learning based Weather Prediction System developed using Python.

The project predicts weather conditions based on:

* Temperature
* Humidity
* Pressure
* Wind Speed
* Rainfall
* City

## Technologies Used

* Python
* Machine Learning
* Flask
* Streamlit
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Features

* 30,000+ generated weather dataset
* Interactive Flask web application
* Streamlit deployment interface
* Input validations
* Browser-based prediction
* Data visualization charts

## Machine Learning Algorithm

* Random Forest Classifier

## How To Run

### Flask App

```bash
python app.py
```

### Streamlit App

```bash
cd streamlit_app
streamlit run streamlit_app.py
```

## Dataset

Generated dataset contains:

* temperature
* humidity
* pressure
* wind_speed
* rainfall
* city
